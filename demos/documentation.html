Options

HelloWeek comes with a few (optional) settings that you can change by passing an object as an argument.

selector

Type: String
Default: .hello-week

lang

Type: String
Default: en

langFolder

Type: String
Default: ./dist/langs/

format

Type: BooleanString
Default: false

weekShort

Type: Boolean
Default: true

monthShort

Type: Boolean
Default: false

multiplePick

Type: Boolean
Default: false

defaultDate

Type: BooleanString
Default: false

todayHighlight

Type: Boolean
Default: true

disablePastDays

Type: Boolean
Default: false

disabledDaysOfWeek

Type: BooleanArray
Default: false

disableDates

Type: BooleanArray
Default: false

daysHighlight v1.2.0new

Type: BooleanArray
Default: false

weekStart v1.0.0

Type: Integer
Default: false

minDate

Type: BooleanSring
Default: false

maxDate

Type: BooleanSring
Default: false
